# Nothing is Isolated — Systems Thinking Lab

A self-learning foundation module for students entering undergraduate visual communication programs.

---

## What this is

A 6-phase self-paced course designed to shift how students see the world — from isolated objects to interconnected systems — before they begin designing anything.

No lectures. No grades. No fixed pace.

---

## Who it is for

17–18 year olds entering 4-year visual communication programs.  
Suitable as a pre-semester or first-semester foundation module.  
Can be run independently or with light facilitation.

## How to deploy

### Option 1 — GitHub Pages (recommended)

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. GitHub will publish `index.md` — add a `_config.yml` if you want Jekyll theming

For clean rendering of the Markdown and SVGs, a minimal Jekyll theme works well:

```yaml
# _config.yml
theme: minima
title: Nothing is Isolated
description: Systems Thinking Lab
```

### Option 2 — Direct Markdown

Share `index.md` directly via any Markdown renderer:
- Notion
- Obsidian
- HackMD
- Any static site generator

### Option 3 — Print

The SVGs are print-ready at A4/A3.  
`visual_blank.svg` is designed to be printed and annotated by hand.

## Phases at a glance

| Phase | Intensity | Shift |
|-------|-----------|-------|
| 1 — SEE | 🟢 Light | Objects → Systems |
| 2 — TRACE | 🟡 Medium | Things → Connections |
| 3 — LOOP | 🔴 Heavy | Linear → Circular |
| 4 — INTERVENE | 🟡 Medium | Predictable → Sensitive |
| 5 — POSITION | 🔴 Heavy | Observer → Participant |
| 6 — DESIGN | 🔴 Heavy | Understanding → Shaping |
